# New Explorer: Desktop News Application

> A Desktop GUI using Tkinter that fetches real-time data via API, reducing search time for users by 40%.

## About
News Explorer is a desktop application developed in Python that provides a streamlined interface for accessing and reading the latest news articles from various categories. 

The application uses Tkinter for its graphical user interface (GUI) and fetches news data through the GoogleNews API, offering users an organized way to stay updated on current events.

## Key Features
* Real-time Updates: Fetches the latest data instantly via API.
* Tab-Based Interface: Browse news by specific regions or topics.
* Categorized Feeds: Includes dedicated sections for Technology, Renewable Energy, and Current Affairs.
* Modern UI: Utilizes `sv_ttk` for a sleek, modern look.
* Scalable Architecture: The structure supports future expansion for additional features.

## Technologies Used
* Language: Python
* GUI: Tkinter, sv_ttk (Theme)
* API: GoogleNews
* Utilities: Threading, Webbrowser
